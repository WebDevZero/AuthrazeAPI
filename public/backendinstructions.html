How to Use This Backend

This backend handles users, messages, and admin actions. Here are the available routes you can visit or use:

  • /users – View all registered users.
  • /messages – View all posted messages.
  • /admin/panel – Admin panel (only accessible if your username is admin).
  • /board – Message board page where users can post and view messages.
  • /delete-user – POST endpoint to delete a user (used by admin panel).
  • /delete-message – POST endpoint to delete a message (only by owner of the message).
  • /backendinstructions – Shows Instructions on how to use backend.
